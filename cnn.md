# 基于CNN+注意力机制的图像去噪

## 一、研究背景与意义

图像去噪是底层视觉任务的基础问题。传统方法如BM3D依赖手工特征，深度学习方法如DnCNN、U-Net通过端到端学习取得更好效果。然而，标准CNN对所有特征通道和空间位置一视同仁，忽略了噪声分布的不均匀性（例如平滑区域噪声明显，纹理区域噪声相对不显著）。引入注意力机制可以让模型自适应地强调信息量大的特征、抑制噪声成分，从而提升去噪质量。

可行性：数据集小（BSD68仅68张灰度图），训练快（CPU 2-4小时，GPU 30分钟），代码开源丰富，创新点明确。

## 二、现有技术不足

| 方法 | 优点 | 不足 |
|------|------|------|
| BM3D | 传统最强 | 计算慢，参数固定 |
| DnCNN | 残差学习，批量归一化 | 所有特征同等对待 |
| U-Net | 多尺度特征融合 | 跳跃连接可能传播噪声 |
| 简单CNN+注意力（已有） | 提升性能 | 注意力模块设计复杂，计算量大 |

核心缺陷：现有方法在加入注意力时，通常直接堆叠SE或CBAM模块，未针对去噪任务设计轻量且有效的注意力机制；且缺乏对"注意力是否真的聚焦于信号而非噪声"的可视化分析。

## 三、本课题创新点（3选1或组合）

1. **轻量级混合注意力模块（LMAB）**：结合通道注意力和空间注意力，但使用深度可分离卷积减少参数量，专为去噪任务优化。
2. **渐进式注意力融合**：在U-Net的编码器和解码器之间，不同层级使用不同强度的注意力，低层保留细节，高层抑制噪声。
3. **残差注意力学习**：让注意力模块学习"噪声残差"，而非直接输出权重，训练更稳定。

> 建议选择创新点1，改动小、效果明显、容易写。

## 四、数据集选用

| 数据集 | 用途 | 规模 | 获取方式 |
|--------|------|------|----------|
| BSD68 | 测试集 | 68张灰度图 | 公开下载 |
| Waterloo Exploration Database | 训练集 | 4744张 | 公开 |
| 或直接用BSD400（400张训练） | 训练+验证 | 400张 | 公开 |

简化方案：仅使用BSD68中的随机分拆（40训练/28测试），外加Set12做额外测试。这样数据集总大小<100MB，任何电脑都能跑。

噪声合成：添加高斯噪声 σ=15, 25, 50（常用三种强度），训练时随机裁剪成50×50 patches。

## 五、基线方法介绍

- **DnCNN**：17层全卷积，残差学习，BN，ReLU。去噪领域的标准基线。
- **U-Net**：编码器-解码器，跳跃连接，无注意力。
- **CBDNet**（可选）：更复杂的噪声估计模型，可作为强对比。

> 为了控制工作量，建议只对比 DnCNN 和 U-Net。

## 六、基本框架及技术细节

### 整体架构图

```text
输入噪声图像 (N×C×H×W)
    ↓
Conv3×3 + ReLU (浅层特征提取)
    ↓
┌─────────────────────────┐
│  多个残差块（每个块包含： │
│    Conv3×3 + BN + ReLU  │
│    + 轻量混合注意力模块   │ ← 创新点
│    + 残差连接            │
└─────────────────────────┘
    ↓
Conv3×3 (重建)
    ↓
残差连接（全局残差：输出 = 输入 - 预测噪声）
    ↓
干净图像
```

### 轻量混合注意力模块（LMAB）细节

```python
# 输入 X: [B, C, H, W]
# 通道注意力部分
avg_pool → FC(降维C/r) → ReLU → FC(升维C) → sigmoid → 通道权重
# 空间注意力部分
Conv1×1 (降维至1) + sigmoid → 空间权重
# 融合
输出 = X * 通道权重 * 空间权重
# 为减少参数量，使用深度可分离卷积替代部分标准卷积
```

### 损失函数

均方误差（MSE）：`L = || y_pred - y_clean ||^2`

可选添加感知损失（VGG特征匹配），但会增加复杂度，不推荐。

## 八、对比实验与消融实验

### 对比实验设计（3个对比方法）

| 方法 | 说明 |
|------|------|
| DnCNN | 标准基线 |
| U-Net | 多尺度基线 |
| Ours (CNN + LMAB) | 你的完整模型 |

评价指标：

- **PSNR (dB)** ↑ 越高越好
- **SSIM** ↑ 越高越好
- **参数量 (M)** ↓ 效率指标
- **推理时间 (ms)** ↓

### 消融实验设计（3组）

| 消融编号 | 说明 | 目的 |
|----------|------|------|
| Ablation A | 去掉注意力模块（退化为普通CNN） | 验证注意力是否有效 |
| Ablation B | 仅通道注意力 + 无空间注意力 | 分析两种注意力的贡献 |
| Ablation C | 使用CBAM替换LMAB | 验证轻量化设计的优越性（或至少不差） |

附加消融（可选）：不同注意力放置位置（仅编码器 vs 仅解码器 vs 全都有）。

## 九、实验结果分析（预期）

预期结论（需要你实际跑出来）：

- Ours在PSNR上比DnCNN高0.5~1.2 dB（σ=25时），比U-Net高0.3~0.8 dB。
- 参数量仅增加约10%（LMAB很轻量），推理时间增加<15%。
- 消融A表明注意力模块贡献了主要的性能提升（移除后PSNR下降0.7 dB）。
- 消融B表明通道注意力和空间注意力都有正向贡献，且协同作用。
- 消融C表明LMAB与CBAM性能接近但参数更少，证明轻量化有效。

可视化分析：

- 展示去噪前后图像对比（选一个细节丰富的图）。
- 展示注意力热力图：将LMAB的输出权重可视化，证明模型更关注边缘/纹理区域，平滑区域权重低（避免过度平滑）。

## 十、研究工作总结

总结你的贡献：

1. 提出了一种轻量级混合注意力模块LMAB，专用于图像去噪，参数量少且有效。
2. 通过对比实验证明CNN+LMAB优于DnCNN和U-Net。
3. 通过消融实验验证了注意力机制的必要性以及各组成部分的贡献。
4. 提供了可视化分析，解释注意力如何帮助去噪。

可扩展讨论：模型在更高噪声（σ=50）下性能下降较快，未来可考虑多尺度噪声估计。

## 代码获取与修改指南

### 推荐的开源代码仓库

- DnCNN (PyTorch)：<https://github.com/cszn/DnCNN> (官方)
- U-Net for denoising：<https://github.com/jaxony/unet-pytorch> (需改输出)
- 带注意力的简化实现：搜 `pytorch image denoising attention`

### 修改步骤（最简单路线）

1. 跑通 DnCNN 官方代码，确认能在你的环境训练。
2. 在 `model.py` 中定义 `LightweightMixedAttention` 模块。
3. 找到 DnCNN 的残差块，在每个块中插入 LMAB（位置：BN+ReLU之后，残差加之前）。
4. 训练三个模型：原始 DnCNN、DnCNN+LMAB、消融A（LMAB输出恒等映射）。
5. 跑测试集，记录PSNR/SSIM，画表。

总代码改动量：约80行。
